✅ Key Features:

1. Add Course to Wishlist
    * Users can bookmark or "heart" a course.
    * Should prevent duplicates.

2. View Wishlist
    * Show all saved courses.
    * Show basic course info.

3. Remove from Wishlist
    * Users can un-save a course.

4. Move to Cart
    * Allow users to directly move a course from wishlist to cart.

5. Wishlist Persistence
    * Always saved to the user’s account (backend, not frontend-only).

6. Check if Already in Wishlist
    Useful to toggle “♥” icons on the frontend.
